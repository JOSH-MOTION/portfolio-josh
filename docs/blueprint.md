# **App Name**: JOSHDEV

## Core Features:

- Portfolio Display: Display a clean and responsive layout showcasing projects, skills, and experience.
- Interactive Navigation: Provide an interactive way to navigate through different sections of the portfolio.
- AI-Powered Content Enhancement: Suggest improvements to the user's portfolio description or project write-ups using a generative AI tool.
- Contact Section: A contact form or link allowing potential employers or collaborators to reach out.
- Clear Content Hierarchy: Use clear headings and structured content to guide visitors through your portfolio.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust, stability, and sophistication.
- Background color: Light Gray (#F5F5F5), near-white to provide a clean, unobtrusive backdrop.
- Accent color: Vibrant Amber (#FFC107) to highlight key elements and calls to action, creating a memorable contrast.
- Headline font: 'Space Grotesk', a geometric sans-serif, to give a techy, scientific, and slightly futuristic feel.
- Body font: 'Inter', a grotesque-style sans-serif for the body text, ensures readability and a modern aesthetic. Pairs well with Space Grotesk.
- Use a consistent set of line icons for skills, technologies, and navigation to add visual interest without overwhelming the design.
- Employ a grid-based layout to organize content in a clear and structured manner, ensuring visual consistency and easy navigation.
- Implement subtle transitions and animations to provide feedback on user interactions, such as button hover effects and smooth scrolling.
